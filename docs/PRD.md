## 产品概述

- **产品名称**：本地项目与文件夹管理后台
- **产品类型**：本地运行的可视化管理后台
- **目标版本**：V1.0
- **目标用户**：个人用户，主要用于管理本机收藏文件、常用文件夹、开发项目、资料项目和长期事项
- **运行环境**：本地电脑，优先支持 macOS，后续可扩展 Windows / Linux
- **技术方向**：Next.js 全栈应用，本地数据库，本地文件系统访问能力

## 核心能力

1. 本地文件 / 文件夹收藏与可视化管理
2. 项目独立记录（名称、路径、状态、标签、笔记等）
3. 全局搜索与多维筛选
4. 本地路径检测与一键打开
5. 标签分类管理
6. 数据本地存储、导入导出
