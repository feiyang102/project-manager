# MVP 范围与里程碑

> 所属 PRD：[本地项目与文件夹管理后台 PRD](../PRD.md)

## 15. MVP 范围

V1.0 MVP 建议优先实现：

1. 项目 CRUD。
2. 文件收藏 CRUD。
3. 标签管理。
4. 项目与文件收藏关联。
5. 本地路径存在性检测。
6. 打开本地路径。
7. 全局搜索。
8. 数据导出 JSON。

## 16. 里程碑

### 阶段 1：基础框架

- 搭建 Next.js 项目结构。
- 建立基础布局、左侧导航、页面路由。
- 接入本地数据库。
- 建立 Project、FileItem、Tag 数据模型。

### 阶段 2：核心管理能力

- 实现项目 CRUD。
- 实现文件收藏 CRUD。
- 实现标签 CRUD。
- 实现列表筛选、排序和搜索。

### 阶段 3：本地文件能力

- 实现路径存在性检测。
- 实现打开本地路径。
- 读取基础文件信息。
- 显示路径失效状态。

### 阶段 4：体验完善

- 增加仪表盘。
- 增加最近访问。
- 增加归档。
- 增加导入导出。
- 优化空状态、确认弹窗和错误提示。
