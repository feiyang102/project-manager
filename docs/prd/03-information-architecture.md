# 信息架构

> 所属 PRD：[本地项目与文件夹管理后台 PRD](../PRD.md)

## 7. 信息架构

### 7.1 一级导航

- 仪表盘
- 项目
- 文件收藏
- 标签
- 最近访问
- 归档
- 设置

### 7.2 主要对象

#### 项目 Project

- 项目名称
- 项目描述
- 项目状态：想法、进行中、暂停、完成、归档
- 项目类型：代码、设计、写作、学习、资料、生活、其他
- 本地路径
- 技术栈 / 工具栈
- 标签
- 优先级
- 开始日期
- 目标日期
- 最近访问时间
- 备注 / 项目笔记
- 关联文件收藏
- 关联链接

#### 文件收藏 File Collection

- 名称
- 类型：文件、文件夹、代码仓库、文档、图片、视频、音频、压缩包、其他
- 本地路径
- 所属项目
- 标签
- 描述
- 收藏时间
- 最近访问时间
- 文件大小
- 是否存在
- 是否归档

#### 标签 Tag

- 标签名称
- 标签颜色
- 使用次数
- 创建时间

#### 链接 Link

- 标题
- URL
- 所属项目
- 描述
- 标签

#### 笔记 Note

- 标题
- 正文
- 所属项目
- 创建时间
- 更新时间
