# 页面规划

> 所属 PRD：[本地项目与文件夹管理后台 PRD](../PRD.md)

## 11. 页面规划

### 11.1 仪表盘 `/`

- 顶部统计区
- 最近访问
- 进行中项目
- 最近添加收藏

### 11.2 项目列表 `/projects`

- 搜索框
- 筛选栏
- 项目列表 / 看板切换
- 新建项目按钮

### 11.3 项目详情 `/projects/[id]`

- 项目基础信息
- 关联文件
- 笔记
- 链接
- 操作历史

### 11.4 文件收藏 `/files`

- 搜索框
- 类型筛选
- 标签筛选
- 收藏列表
- 添加文件 / 文件夹按钮

### 11.5 标签 `/tags`

- 标签列表
- 标签创建和编辑
- 标签使用统计

### 11.6 最近访问 `/recent`

- 最近项目
- 最近文件收藏

### 11.7 归档 `/archive`

- 已归档项目
- 已归档文件收藏

### 11.8 设置 `/settings`

- 数据存储
- 导入导出
- 默认目录
- 应用信息
