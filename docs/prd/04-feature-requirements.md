# 功能需求

> 所属 PRD：[本地项目与文件夹管理后台 PRD](../PRD.md)

## 8. 功能需求

### 8.1 仪表盘

#### 功能描述

仪表盘用于展示用户当前最需要关注的项目和文件资产概览。

#### 需求列表

- 展示项目数量、收藏文件数量、进行中项目数量、归档数量。
- 展示最近访问的项目和文件夹。
- 展示进行中的项目列表。
- 展示最近添加的收藏项。
- 支持点击项目或文件收藏进入详情页。

#### 验收标准

- 用户打开应用后，可以在 3 秒内看到核心统计信息。
- 用户可以从仪表盘进入任意最近访问项目或文件收藏。

### 8.2 项目列表

#### 功能描述

集中展示所有项目，支持搜索、筛选、排序和批量管理。

#### 需求列表

- 支持列表视图和看板视图。
- 支持按状态、类型、标签、优先级筛选。
- 支持按创建时间、更新时间、最近访问时间排序。
- 支持新建、编辑、归档、删除项目。
- 支持显示项目关联路径是否仍然存在。

#### 验收标准

- 用户可以在项目列表中快速找到目标项目。
- 编辑项保存后，列表和详情页数据同步更新。

### 8.3 项目详情

#### 功能描述

项目详情页是单个项目的管理中心。

#### 需求列表

- 展示项目基础信息。
- 支持编辑项目名称、描述、状态、类型、标签、路径和备注。
- 支持关联多个文件收藏。
- 支持添加项目笔记。
- 支持添加外部链接。
- 支持记录项目状态变化历史。
- 支持打开本地项目路径。

#### 验收标准

- 项目详情页可以完整表达一个项目的当前状态和相关资料。
- 用户可以从项目详情页打开本地路径。

### 8.4 文件收藏

#### 功能描述

管理用户收藏的本地文件和文件夹。

#### 需求列表

- 支持添加文件或文件夹路径。
- 支持手动输入路径和通过文件选择器选择路径。
- 支持为收藏项添加名称、描述、标签、所属项目。
- 支持检测路径是否存在。
- 支持打开文件或文件夹。
- 支持复制路径。
- 支持归档和删除收藏项。
- 支持按类型、标签、所属项目、是否存在筛选。

#### 验收标准

- 用户可以成功添加一个本地文件夹收藏。
- 收藏项路径失效时，界面能清晰提示。

### 8.5 标签管理

#### 功能描述

标签用于跨项目和文件收藏建立灵活分类。

#### 需求列表

- 支持创建、编辑、删除标签。
- 支持设置标签颜色。
- 支持查看标签关联的项目和收藏项数量。
- 删除标签时，不删除关联项目或文件，只移除标签关系。

#### 验收标准

- 标签变化后，项目和文件收藏中的标签展示同步更新。

### 8.6 全局搜索

#### 功能描述

提供跨项目、文件收藏、标签、笔记和链接的统一搜索入口。

#### 需求列表

- 支持关键词搜索。
- 支持按对象类型筛选搜索结果。
- 搜索范围包括名称、描述、路径、标签、笔记标题和链接标题。
- 搜索结果支持跳转到详情页。

#### 验收标准

- 输入关键词后可以在 1 秒内返回本地数据库结果。
- 搜索结果点击后跳转准确。

### 8.7 最近访问

#### 功能描述

记录用户最近打开或查看过的项目和收藏项。

#### 需求列表

- 自动记录访问项目详情、打开路径、查看收藏项的时间。
- 最近访问列表按时间倒序排列。
- 支持清空最近访问记录。

#### 验收标准

- 用户打开一个项目后，该项目出现在最近访问列表顶部。

### 8.8 归档

#### 功能描述

用于收纳不再活跃但需要保留记录的项目和收藏项。

#### 需求列表

- 项目和文件收藏均可归档。
- 归档项默认不出现在主列表中。
- 支持从归档中恢复。

#### 验收标准

- 归档后主列表隐藏，归档页可见。
- 恢复后主列表重新可见。

### 8.9 设置

#### 功能描述

管理应用本地配置。

#### 需求列表

- 设置默认扫描目录。
- 设置默认项目类型和状态。
- 设置数据备份位置。
- 支持导出数据为 JSON。
- 支持从 JSON 导入数据。
- 显示数据库位置和应用版本。

#### 验收标准

- 用户可以导出当前所有项目和收藏数据。
- 用户可以查看本地数据存储位置。
