# 验收清单与后续增强

> 所属 PRD：[本地项目与文件夹管理后台 PRD](../PRD.md)

## 17. 验收清单

- 可以创建、编辑、删除和归档项目。
- 可以收藏本地文件或文件夹路径。
- 可以为项目和收藏项添加标签。
- 可以通过搜索找到项目和收藏项。
- 可以进入项目详情查看关联资料。
- 可以检测本地路径是否存在。
- 可以从界面打开本地路径。
- 可以导出本地数据。
- 不会在用户未确认时修改或删除真实文件。

## 18. 后续增强想法

- 自动扫描常用目录。
- Git 仓库识别和状态展示。
- 项目启动命令记录和一键打开终端。
- README / package.json 自动解析。
- AI 生成项目摘要。
- 文件夹内容索引。
- 截图、封面图和项目图标。
- 数据备份与迁移。
- 快捷键启动器。
- 浏览器书签导入。
