# Tools - 项目工具集

本目录存放项目开发过程中使用的各类工具脚本。

## 工具列表

| 工具 | 说明 | 详情 |
|------|------|------|
| [generate-icons](./generate-icons) | 图标生成工具，从 SVG/PNG 源图自动生成 Tauri 桌面应用全套图标（PNG/ICO/ICNS），支持自动裁剪留白、居中填充、透明背景修复 | [查看详情](./generate-icons/README.md) |

## 新增工具

新增工具时请：
1. 在本目录下创建以工具名命名的子目录
2. 将工具脚本放入子目录
3. 在子目录中创建 `README.md` 编写详细使用说明
4. 在上方表格中登记工具信息
